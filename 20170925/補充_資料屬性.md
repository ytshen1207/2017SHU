補充 - 資料屬性
================

資料屬性
--------

``` r
1; 20.0; 3e2  # 數值
```

    ## [1] 1

    ## [1] 20

    ## [1] 300

``` r
class(1)
```

    ## [1] "numeric"

``` r
"stat"  # 文字
```

    ## [1] "stat"

``` r
class("stat")
```

    ## [1] "character"

``` r
TRUE; T; FALSE; F  # 邏輯真假值
```

    ## [1] TRUE

    ## [1] TRUE

    ## [1] FALSE

    ## [1] FALSE

``` r
class(T)
```

    ## [1] "logical"

-   若將多個語句寫在同一行時可用分號(;)代表語句結束，否則一般情況下可省略。
-   使用成對的雙引號(")或單引號(')表示字串。
-   TRUE和FALSE為R的保留字，T和F則不是，使用上需小心。
