補充 - 指標系統
================

向量及矩陣的指標用法
--------------------

``` r
x = 1:5; y = x + 1
x[1]
```

    ## [1] 1

``` r
y[c(2, 3, 4)]  # 等同於y[2:4]
```

    ## [1] 3 4 5

``` r
x >= 3
```

    ## [1] FALSE FALSE  TRUE  TRUE  TRUE

``` r
y[x >= 3]
```

    ## [1] 4 5 6

``` r
x[x <= 2 | y == 5]  # 且(&, &&)，或(|, ||)
```

    ## [1] 1 2 4

``` r
length(x[x < 4])
```

    ## [1] 3

``` r
sum(y[y != 6])  # 不等號!=
```

    ## [1] 14

``` r
x[-1]
```

    ## [1] 2 3 4 5

``` r
y[-(2:4)]
```

    ## [1] 2 6

``` r
X = rbind(x, y)  # row bind
Y = cbind(x, y)  # column bind
X[1, 5]; X[, 2]; X[, c(1, 3)]; Y[2 ,]; Y[2:4, -1]; Y[-1, -2]
```

    ## x 
    ## 5

    ## x y 
    ## 2 3

    ##   [,1] [,2]
    ## x    1    3
    ## y    2    4

    ## x y 
    ## 2 3

    ## [1] 3 4 5

    ## [1] 2 3 4 5

-   &, | 與 &&, || 的差別在於前者適用於向量的條件判斷。
